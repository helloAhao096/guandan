# Spec: action-buttons

## ADDED Requirements

### Requirement: 结果按钮与确认/回退

- **WHEN** 渲染操作区
- **THEN** 三个结果按钮（头游+二游、头游+三游、头游+末游），段落式；单侧选择（点击左侧=红方赢，右侧=蓝方赢）；回退、确认始终显示；确认未选择时禁用（灰）

#### Scenario: 确认按钮尺寸

- **WHEN** 查看回退与确认按钮
- **THEN** 两钮为方形，两钮总宽度与单个结果按钮（头游+二游）一致

### Requirement: 箭头独立

- **WHEN** 渲染左右区域
- **THEN** 红方◀、▶蓝方与按钮分离，箭头指向中央
